# 카카오 로그인 KOE205 에러 해결 방법

KOE205 에러는 카카오 개발자 콘솔 설정 문제로 발생합니다. 다음을 확인하세요:

## 1. 카카오 로그인 사용 설정 확인
- 카카오 개발자 콘솔 > 내 애플리케이션 > [카카오 로그인] > [사용 설정]
- **상태를 ON으로 설정**되어 있는지 확인

## 2. 리다이렉트 URI 확인 (가장 중요!)
- 카카오 개발자 콘솔 > 내 애플리케이션 > [카카오 로그인] > [일반] > [리다이렉트 URI]
- 다음 URI가 **정확히** 등록되어 있어야 합니다:
  ```
  http://localhost:3000/api/auth/kakao/callback
  ```
- **주의사항:**
  - HTTP와 HTTPS는 다르게 인식됩니다
  - 경로 끝에 `/`가 있으면 안 됩니다
  - 대소문자를 구분합니다
  - 정확히 일치해야 합니다

## 3. 동의항목 설정 확인
- 카카오 개발자 콘솔 > 내 애플리케이션 > [카카오 로그인] > [동의항목]
- 다음 항목이 **활성화**되어 있어야 합니다:
  - 닉네임 (필수 또는 선택)
  - 프로필 사진 (선택)
  - 카카오계정(이메일) (선택)

## 4. Client ID 확인
- `.env` 파일의 `KAKAO_CLIENT_ID`가 올바른지 확인
- 카카오 개발자 콘솔 > 내 애플리케이션 > [앱 키] > REST API 키

## 5. Client Secret 확인 (사용하는 경우)
- `.env` 파일의 `KAKAO_CLIENT_SECRET`이 올바른지 확인
- Client Secret을 사용하는 경우, [카카오 로그인] > [일반] > [Client Secret]에서 상태가 "사용함"인지 확인

## 빠른 체크리스트
- [ ] 카카오 로그인 사용 설정이 ON인가?
- [ ] 리다이렉트 URI가 정확히 일치하는가?
- [ ] 동의항목이 설정되어 있는가?
- [ ] Client ID가 올바른가?
- [ ] Client Secret이 올바른가? (사용하는 경우)
